# Key Findings

- **Moderate positive association** between diet score and overall cognition (r = 0.367)
- **Moderate positive association** between diet score and language (r = 0.353)
- **Moderate positive association** between diet score and logical reasoning (r = 0.336)
- **Weak associations** for Stroop and memory (near zero correlations)
- **Explained variance** from Diet–Cognition correlation: r² ≈ 0.135 (13.5%)
- Effect size falls within the **moderate** range (0.30–0.50)
