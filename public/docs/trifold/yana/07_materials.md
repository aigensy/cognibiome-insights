# Materials

**Data Source**
- De-identified adolescent pilot dataset (n = 66)
- Composite diet quality score
- Cognitive outcomes: Stroop, memory recall, language, logical reasoning, overall composite

**Implementation**
- Deterministic simulator implementing: Diet (D) → Microbiome Proxies (X) → Metabolite Proxies (M) → Cognition Proxies (Y)
- Replaceable parameter artifacts used for modeled layers
- SHA-256 run hash for reproducibility

**Tools**
- Data analysis and visualization tools (software-based)
- No wet lab materials (computational project)
