# Modeling Framework

## A Deterministic Computational Framework

Integrating **Diet**, **Microbiome proxies**, and **Cognitive outcomes**.

### Layered Pipeline

**Measured pilot layer (real data):**
- Diet score
- Cognitive tests (Stroop, memory, language, logical reasoning)

**Modeled proxy layers (simulated):**
- Diet inputs (D)
- Microbiome proxies (X)
- Metabolite proxies (M)
- Cognition proxies (Y)

### Core Sequence

**Diet (D) → Microbiome Proxies (X) → Metabolite Proxies (M) → Cognition Proxies (Y)**

### Reproducibility

Runs are deterministic and produce a **SHA-256 run hash** to support auditability.
