# Limitations

- Observational pilot sample (n = 66)
- Self-reported dietary data
- No direct biological measurements (microbiome/metabolites were not measured in the pilot)
- Confounding variables were not experimentally controlled
- Modeled intermediate layers use simplified assumptions
- Not adjusted for multiple comparisons
- This version demonstrates the workflow using modeled proxy layers; robust mechanistic validation requires paired microbiome/metabolomics measurements.
