# Conclusion

In a de-identified adolescent pilot sample (n = 66), higher diet quality scores were positively correlated with overall cognitive performance (r = 0.367). Positive associations were observed for language (r = 0.353) and logical reasoning (r = 0.336), while Stroop and memory showed weak associations.

These results provide an association signal, but underlying biological mechanisms were not measured in this dataset. Therefore, this project's main contribution is an educational, reproducible modeling framework that demonstrates how to structure and study diet → microbiome → metabolites → cognition problems when sufficient paired data and computing resources are available.

The simulator supports structured hypothesis generation and teaching. It is not diagnostic, not medical advice, and does not prove causality.
