# Materials

**Data Source**
- De-identified adolescent pilot dataset (n = 66)
- Composite diet quality score
- Cognitive outcomes: Stroop, memory recall, language, logical reasoning, overall composite

**Implementation**
- CogniBiome Insights deterministic educational simulator implementing:
  Diet (D) → Microbiome Proxies (X) → Metabolite Proxies (M) → Cognition Proxies (Y)
  *(Designed to run without internet during judging for demo reliability.)*

**Tools**
- Software-based analysis and visualization tools
- No wet lab materials (computational project)
